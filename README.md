
## Name:GoodHabit - Habit Tracker Website
Live Site URL:https:https://phenomenal-mooncake-e6698e.netlify.app/addhabits

##  Features

-  User authentication with login & signup system  
-  Add, edit, and delete daily habits  
-  Track habit completion with an easy-to-use
-  Progress tracking with streak system  
-  Responsive UI for mobile and desktop  
-  Secure backend API for handling user data  
-  Modern UI built with React + Tailwind
